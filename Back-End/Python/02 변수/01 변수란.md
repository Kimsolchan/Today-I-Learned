# 변수(Variable)
## 변수란
- 값을 가리키고 있는 것
```
컴퓨터는 값(data)를 메모리에 저장하는데 숫자 형태로 되어있어 

사람이 기억하기 힘들다. 그렇기 때문에 변수를 사용하여 값을 저장하고 있는 메모리의 주소를 

대신하여 사용하는 것이다.

메모리 : 컴퓨터가 프로그램에서 사용하는 데이터를 저장하는 공간
```

## 변수를 선언하는 방법
- 변수를 만들땐 =(assignment)기호를 사용한다.
```
변수이름 = 변수에 저장할 값
```

### 변수 선언 예시
```python
num = 100
name = "python"
print(num , name)

100 python
```
